# 介绍
CCP-Server是一个低代码开发平台。提供强大的接口开发逻辑，支持项目中常见接口的逻辑编排实现。

## 发布版本
|             资源名              | 时间 |
|:----------------------------:| :----: |
|             1.0           | 敬请期待  |

### 特点

* 高性能
* 低代码
* 多协议支持（Http，GRPC）
* 微服务，单机一体
* 接口编排
* 多数据源支持
* 插件扩展
* 接口文档自动生成
* 提供Dashboard交互面板
